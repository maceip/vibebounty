How do I serve it locally? 
// MLX (Apple silicon) — the tuned adapter:
mlx_lm.server --model WeiboAI/VibeThinker-3B \
  --adapter-path adapters --port 8080

// Ollama:
OLLAMA_ORIGINS='*' ollama serve   # then: ollama run <model>

// vLLM:
vllm serve <model> --port 8080

Tip: allow CORS from this origin so the browser can reach your endpoint (MLX/vLLM expose --cors; Ollama uses OLLAMA_ORIGINS).
